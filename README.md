# space-monitor
    A project to track space related events

# Project Overview
    This project uses the NASA open API's. The events that are tracked are coronal mass ejections, coronal mass ejection analysis, and geomagnetic storms. The Nasa astronomy picture of the day will also be displayed on the website

    The goal of this project is to learn solid backend fundamentals with minimal AI assistance. I am passionate about technology and science, this project is meant to be an engaging way to learn, showcase my development skills, and learn a little about space along the way
